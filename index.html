Política de Privacidade

Última atualização: 23 de julho de 2025

O bot Nishiki valoriza sua privacidade. Esta política descreve como coletamos e usamos dados no Discord.

1. Quais dados coletamos?

  • ID de usuário e servidor
  • Preferências e configurações do servidor
  • Interações com comandos

2. Como usamos os dados

Utilizamos os dados para:

  • Oferecer comandos de economia, diversão, utilidades e moderação
  • Melhorar a experiência dos usuários

3. Compartilhamento

Não compartilhamos seus dados com terceiros. Os dados são usados exclusivamente para funcionalidades do bot.

4. Exclusão de dados

Você pode solicitar a exclusão dos seus dados a qualquer momento entrando em contato com o desenvolvedor.

5. Mudanças nesta política

Esta política pode ser atualizada periodicamente. Verifique sempre a versão mais recente.
