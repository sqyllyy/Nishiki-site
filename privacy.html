Política de Privacidade

Última atualização: 23 de julho de 2025

Esta Política de Privacidade descreve como o bot Nishiki coleta, armazena e utiliza informações dentro do Discord, oferecendo comandos de economia, diversão, utilidades e moderação.

1. Informações Coletadas

  • ID de usuário e ID de servidor para identificação e funcionalidades internas;
  • Configurações personalizadas salvas por servidor ou usuário;
  • Dados estatísticos de uso de comandos para melhorar a experiência.

2. Como Usamos as Informações

  • Para fornecer recursos como economia e moderação;
  • Para registrar pontuações e interações em comandos de diversão;
  • Para manter configurações persistentes por servidor e por usuário.

3. Compartilhamento de Dados

O bot Nishiki não compartilha dados coletados com terceiros. Todos os dados são utilizados exclusivamente para o funcionamento interno do bot.

4. Segurança

Medidas técnicas básicas são adotadas para proteger os dados armazenados, evitando acesso não autorizado.

5. Exclusão de Dados

Qualquer usuário ou administrador de servidor pode solicitar a exclusão completa de seus dados armazenados entrando em contato com o desenvolvedor.

6. Alterações nesta Política

Esta política pode ser alterada sem aviso prévio. Recomendamos revisá-la periodicamente.

7. Contato

Em caso de dúvidas ou solicitações relacionadas à privacidade, entre em contato pelo Discord com o desenvolvedor do bot Nishiki.
